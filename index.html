// ═══════════════════════════════════════════════════ // Shipment + Transfers Live API — Apps Script // انسخ هذا الكود في ملف الشحنات // ═══════════════════════════════════════════════════ const TRANSFER_SHEET_ID = 'TRANSFER_SHEET_ID_PLACEHOLDER'; function doGet(e) { try { const action = e.parameter.action || 'all'; if (action === 'shipments') return jsonResponse(getShipments()); if (action === 'transfers') return jsonResponse(getTransfers()); return jsonResponse(getMerged()); } catch(err) { return jsonResponse({ error: err.toString() }); } } function jsonResponse(data) { const output = ContentService.createTextOutput(JSON.stringify(data)); output.setMimeType(ContentService.MimeType.JSON); return output; } // ── جلب الشحنات ── function getShipments() { const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheets = ss.getSheets(); const result = []; const SHIP_COLS = { roq_invoice: 'ROQ / سعر و رقم الفاتورة', containers_no: 'رقم الحاوية CONTAINERS NO', containers_count: 'عدد الحاويات COUNTS', qty: 'الكمية QTY', line: 'أسم الخط LINE', details: 'التفاصيل', ship_date: 'تاريخ الشحن', eta_expected: 'تاريخ الوصول المفترض', eta_umqasr: 'أم قصر ETA', ata_baghdad: 'بغداد ATA', bl: 'BL رقم البوليصة', shipper: 'الشاحن', po: 'اودو', warehouse_invoice: 'فاتورة المخزن', manifest: 'منفيست / رقم البيان الكمركي', roq: 'ROQ', pi: 'PI', hs_code: 'hs code', pre_customs: 'البيان المسبق', }; sheets.forEach(sheet => { const name = sheet.getName(); if (name === 'Sheet1' || name.includes('Template')) return; const data = sheet.getDataRange().getValues(); if (!data.length) return; const headers = data[0].map(h => String(h).trim()); const colIdx = {}; Object.entries(SHIP_COLS).forEach(([key, label]) => { let i; if (key === 'po') { // مطابقة مرنة لعمود P.O — قد يكون اسمه 'اودو' أو 'P.O ( odoo )' أو 'PO' حسب الشيت i = headers.findIndex(h => h.includes('اودو')); if (i < 0) i = headers.findIndex(h => h.toLowerCase().includes('odoo')); if (i < 0) i = headers.findIndex(h => h.includes('P.O') || h.trim() === 'PO'); } else { i = headers.findIndex(h => h === label || h.includes(label.slice(0,8))); } if (i >= 0) colIdx[key] = i; }); // Note column (varies by sheet) const noteIdx = headers.findIndex(h => h.includes('ملاحظة') || h.includes('ملاحظات')); const senderIdx = headers.findIndex(h => h.includes('sender bank') || h.includes('Bank')); const amtIdx = headers.findIndex(h => h.includes('Amount') || h.includes('$ Amount')); for (let i = 1; i < data.length; i++) { const row = data[i]; const po = colIdx.po !== undefined ? String(row[colIdx.po]).trim() : ''; if (!po || !po.startsWith('P')) continue; const obj = { month: name }; Object.entries(colIdx).forEach(([key, idx]) => { obj[key] = String(row[idx] ?? '').trim(); }); if (noteIdx >= 0) obj.notes = String(row[noteIdx] ?? '').trim(); if (senderIdx >= 0) obj.ship_bank = String(row[senderIdx] ?? '').trim(); if (amtIdx >= 0) obj.ship_amount = String(row[amtIdx] ?? '').trim(); result.push(obj); } }); return result; } // ── جلب الحوالات ── function getTransfers() { const ss2 = SpreadsheetApp.openById(TRANSFER_SHEET_ID); const sheets = ss2.getSheets(); const result = []; const HEADER_ROW_MAP = { 'وهج الماسة': 5, 'الفالة ': 5, 'زهور الامل ': 1 }; sheets.forEach(sheet => { const name = sheet.getName().trim(); const headerRow = HEADER_ROW_MAP[sheet.getName()] || 5; const data = sheet.getDataRange().getValues(); if (data.length < headerRow + 1) return; const headers = data[headerRow - 1].map(h => String(h).trim()); const poIdx = headers.findIndex(h => h.includes('P.O') || h.toLowerCase().includes('odoo')); const fromIdx = headers.findIndex(h => h.includes('From')); const bankIdx = headers.findIndex(h => h.trim() === 'Bank' || h.trim() === 'Bank '); const toIdx = headers.findIndex(h => h.includes('To CO')); const benfIdx = headers.findIndex(h => h.includes('Beneficiary')); const invIdx = headers.findIndex(h => h.includes('Invoice No')); const amtIdx = headers.findIndex(h => h.includes('Amount')); const dateIdx = headers.findIndex(h => h.trim() === 'Date'); const notesIdx = headers.findIndex(h => h.trim().toLowerCase() === 'notes'); const notes2Idx = headers.findLastIndex(h => h.trim().toLowerCase().includes('notes')); const swiftIdx = headers.findIndex(h => h.includes('Receive Swift')); for (let i = headerRow; i < data.length; i++) { const row = data[i]; if (poIdx < 0) continue; const po = String(row[poIdx] ?? '').replace(/\n/g,'').trim(); if (!po || !po.startsWith('P')) continue; const amt = parseFloat(String(row[amtIdx] ?? '').replace(/[^0-9.]/g,'')) || 0; const notesArr = []; if (notesIdx >= 0 && row[notesIdx]) notesArr.push(String(row[notesIdx]).trim()); if (notes2Idx >= 0 && notes2Idx !== notesIdx && row[notes2Idx]) notesArr.push(String(row[notes2Idx]).trim()); result.push({ po, from_co: fromIdx >= 0 ? String(row[fromIdx] ?? '').trim() : '', bank: bankIdx >= 0 ? String(row[bankIdx] ?? '').trim() : '', to_co: toIdx >= 0 ? String(row[toIdx] ?? '').trim() : '', benef_bank: benfIdx >= 0 ? String(row[benfIdx] ?? '').trim() : '', invoice_no: invIdx >= 0 ? String(row[invIdx] ?? '').trim() : '', amount_num: amt, amount_raw: String(row[amtIdx] ?? '').trim(), date: dateIdx >= 0 ? String(row[dateIdx] ?? '').trim() : '', notes: notesArr.join(' | '), receive_swift: swiftIdx >= 0 ? String(row[swiftIdx] ?? '').trim() : '', sheet: name, }); } }); return result; } // ── دمج الشحنات + الحوالات ── function getMerged() { const ships = getShipments(); const transfers = getTransfers(); // Group transfers by PO const tByPO = {}; transfers.forEach(t => { if (!tByPO[t.po]) tByPO[t.po] = []; tByPO[t.po].push(t); }); return ships.map(ship => { const trans = tByPO[ship.po] || []; const total = trans.reduce((s, t) => s + t.amount_num, 0); const banks = [...new Set(trans.map(t => t.bank).filter(Boolean))]; const from_cos = [...new Set(trans.map(t => t.from_co).filter(Boolean))]; const validSwifts = trans .map(t => t.receive_swift) .filter(s => s && s.length > 4 && !s.includes('قيد') && !s.includes('الغيت')) .sort(); const latest_swift = validSwifts[validSwifts.length - 1] || ship.ship_swift || ''; return { ...ship, bank_sender: banks[0] || '', all_banks: banks, from_cos, transfer_total: total, swift_latest: latest_swift, installment_count: trans.length, installments: trans.length > 1, transfers: trans.map((t, i) => ({ idx: i + 1, ...t })), }; }); }
